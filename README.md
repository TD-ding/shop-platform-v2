# Shop Platform 购物平台

一个前后端交互的购物平台，支持三种用户角色：管理员、超级用户、普通用户。

## 功能

- 用户注册/登录，JWT 认证
- 管理员：商品管理、用户管理、订单管理
- 超级用户：专属折扣、优先购买、特殊商品
- 普通用户：浏览商品、购物车、下单、查看订单

## 技术栈

- 前端：HTML / CSS / JavaScript
- 后端：Node.js + Express
- 数据库：SQLite
- 认证：JWT Token
