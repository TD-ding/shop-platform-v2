# 协作开发日志 - Shop Platform 购物平台

## 项目概述
一个前后端交互的购物平台，支持管理员、超级用户、普通用户三种角色。

**技术栈**：Node.js + Express + SQLite + JWT + HTML/CSS/JS

**仓库**：https://github.com/TD-ding/shop-platform-v2

---

## 迭代记录

### 第1轮：feat - 初始版本
**PR**: #1
**模糊化输入**：
> "我想做一个购物平台！前后端都要有，用 Node.js + Express 做后端，前端用 HTML/CSS/JavaScript，数据库用 SQLite，认证用 JWT。要支持三种用户角色：管理员、超级用户、普通用户。"
